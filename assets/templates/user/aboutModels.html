Densenet Model
High Accuracy - 97.98 %

DenseNet is a deep learning architecture where each layer is directly connected to every other layer, enabling better feature propagation and reuse. Its unique feature is this dense connectivity, which reduces the number of parameters and improves efficiency, leading to more accurate image recognition.

Mobilenet Model
Moderate Accuracy - 97.43 %

MobileNet is a neural network optimized for mobile and low-resource devices, designed for efficient image classification and detection. Its unique feature is the use of depthwise separable convolutions, which drastically reduce computation and model size while maintaining high accuracy.

Inception Model
Low Accuracy - 94.64 %

Inception is a deep learning architecture that extends the concept of depthwise separable convolutions, offering a more complex and flexible model for image classification tasks. Its unique feature is the incorporation of residual connections between layers, which enhances training efficiency and accuracy
